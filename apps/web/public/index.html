How It Works

Four-step cryptographic workflow from intent submission to atomic settlement

1

Submit Encrypted Intent

Client encrypts order parameters (side, size, limits) with ChaCha20-Poly1305. Intent commitment posted on-chain with escrowed collateral.

2

Arcium MPC Matching

Multi-party computation cluster decrypts intents inside secure enclave. Matches orders via continuous double auction without revealing parameters.

3

Bundle Generation

Arcium signs settlement bundle containing execution prices and fill amounts. Cryptographic proof prevents replay and manipulation.

4

Atomic On-Chain Settlement

Solana program validates bundle signature, checks oracle bounds, executes SPL token transfers via CPI. Single transaction. Zero trust assumptions.
